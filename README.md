## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn
- MongoDB (local or cloud instance)

### 1. Clone the Repository

````bash
git clone https://github.com/konlin008/LMS.git
cd lms

To run the backend:

```bash
cd server
npm install
npm run dev
````

#### Frontend

To run the frontend:

```bash
cd client
npm install
npm run dev
```
